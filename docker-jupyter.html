Docker lab environment

Posted on Tue 19 April 2016 in vitalab

I have made a docker container for our lab. Here the link to the docker hub. It's my first try making a usefull container so any comment and PR will be apreciated.

This container does not support any GPU library right now but maybe in the future, who know.

It's based on archlinux(because the official jupyter fail to build) you can write in the latest (almost) python 2 & 3. The list of all the library include in it is listed on the repository.

But there is a rough summary of well known library:

  • scikit-learn
  • matplotlib
  • scikit-image
  • pandas
  • theano
  • numpy

This ones are more lab specific:

  • nibabel
  • h5py

All this library are available from python 2 & 3 inside the jupyter environment.

Enjoy!
